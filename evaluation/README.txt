This code is provided to help gather statistics on your results on the
scenebackgroundmodeling.net dataset.

This code is tested on Python 2.7

Call "python UTILITY.py datasetPath yourresultPath"
ex :  "python processFolder C:\dataset C:\results"

	*** This code calculates 6 metrics, namely (1)AGE, (2)pEPs, (3)pCEPS, (4)MSSSIM, (5)PSNR, (6)CQM.

A 'cm.txt' file will be created to save the metrics.
